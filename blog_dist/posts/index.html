Managing dotfiles with Nix (on NixOS)

This tutorial will be using Flake, and HomeManager. Based on my dotfiles Step 1: Enabling nix flakes in /etc/nixos/configuration.nix Flakes are not enabled by default on NixOS, so you need to enable them. We can do this by editing /etc/nixos/configuration.nix. At the top of the file add the following lines: nix.package = pkgs.nixVersions.stable; # Use the latest stable version of Nix nix.extraOptions = '' experimental-features = nix-command flakes ''; # Enables nix flakes, and the nix command Step 2: Creating a flake.
